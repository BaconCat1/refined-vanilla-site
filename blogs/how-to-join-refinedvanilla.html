How to Join RefinedVanilla

Guide • January 2026
By: RefinedVanilla Team

Looking for a vanilla anarchy Minecraft server that preserves real game mechanics and bans hacked clients? RefinedVanilla is one of the best vanilla anarchy Minecraft servers available, and joining takes just a few minutes. This guide walks you through connecting on both Java Edition and Bedrock Edition, what to expect when you first log in, and tips to help you survive your first few hours.

Before You Join

Before connecting, there are a few things to know about RefinedVanilla:

  • This is a Minecraft anarchy server no hacks — hacked clients are strictly banned. Only approved client mods are allowed. Check the allowed mods list before installing anything.
  • There are no gameplay plugins — no /home, no /tpa, no land claims, no chest protection.
  • PvP is enabled everywhere. There are no safe zones.
  • Griefing and raiding are part of the game. Hide your base well.
  • Read the full server rules so you know what is and is not allowed.

How to Join on Java Edition

Connecting to RefinedVanilla on Java Edition is straightforward. Follow these steps:

  1. Open Minecraft Java Edition and make sure you are running version 1.21.11. You can select the correct version from the Minecraft Launcher by editing your installation profile.
  2. Click "Multiplayer" from the main menu.
  3. Click "Add Server" to add RefinedVanilla to your server list.
  4. In the Server Name field, type RefinedVanilla (or anything you like).
  5. In the Server Address field, type refinedvanilla.net
  6. Click "Done" to save the server.
  7. Select RefinedVanilla from your server list and click "Join Server".

That is it. You do not need to install any mods to join—the server is fully compatible with a vanilla Minecraft client. If you want to use approved performance or quality-of-life mods, check the allowed mods page for the complete list.

How to Join on Bedrock Edition

RefinedVanilla supports Bedrock Edition players as well, so you can connect from Windows 10/11, Xbox, PlayStation, Nintendo Switch, iOS, or Android. Here is how:

  1. Open Minecraft Bedrock Edition and make sure your game is updated to the latest version.
  2. Go to the "Play" tab and select the "Servers" section.
  3. Scroll to the bottom and click "Add Server".
  4. In the Server Name field, type RefinedVanilla.
  5. In the Server Address field, type refinedvanilla.net
  6. In the Port field, type 19132
  7. Click "Save" and then select the server to join.

Console players (Xbox, PlayStation, Switch): Some consoles restrict connections to non-featured servers. You may need to use a workaround such as a DNS redirect or a companion app to connect to custom server addresses. Search for "how to join custom Bedrock servers on [your platform]" for platform-specific instructions, or ask for help in our Discord server.

What to Expect When You First Join

When you spawn into RefinedVanilla for the first time, you will be in the overworld at or near the world spawn point. Here is what you should expect:

  • Spawn may be chaotic: On any anarchy server, the area around spawn tends to be heavily modified. You may see craters, lava, obsidian structures, or other signs of player activity. This is normal.
  • Get moving immediately: Do not linger at spawn. Other players may be nearby, and PvP is enabled everywhere. Pick a direction and start walking, running, or boating away from spawn as fast as possible.
  • No commands to help you: There is no /home, /spawn, /tpa, or /sethome. Navigation is done the old-fashioned way—coordinates, landmarks, and the F3 debug screen.
  • The world is real vanilla: Every mechanic works exactly like single-player. The server runs custom server software based on Fabric, designed to be as vanilla as possible with multithreading to prevent lag. Your farms, redstone, and contraptions will work here.

Tips for New Players

Surviving on a vanilla anarchy server takes a different mindset than regular survival. Here are some tips to help you get started:

1. Travel Far from Spawn

The most important thing you can do as a new player is put distance between yourself and spawn. Travel at least 10,000 blocks in the overworld before setting up a base. The further out you go, the less likely other players are to stumble across your builds. Use the Nether to travel faster—every block in the Nether equals eight blocks in the overworld.

2. Keep Your Base Coordinates Secret

Never share your base coordinates with anyone you do not fully trust. On an anarchy server, leaked coordinates mean your base will eventually be found and raided. Avoid taking screenshots that show your coordinates in the F3 screen, and be careful about sharing images that reveal recognizable terrain features.

3. Build Underground or Hidden

Surface bases are easy to spot, especially from high vantage points or through terrain exploration. Underground bases are much harder to find. Dig into a mountainside, build beneath an ocean, or carve out space in a cave system far from any trail.

4. Gear Up Before Engaging in PvP

Do not pick fights until you have full diamond or netherite armor with enchantments. PvP on RefinedVanilla uses vanilla combat mechanics—no hacks, no unfair advantages. Practice crystal PvP and sword combat in single-player if you are not confident in your skills.

5. Use Approved Mods for Quality of Life

You do not have to play with a completely bare client. Mods like Sodium (for performance), Litematica (for building schematics), and approved minimaps can make your experience much smoother. Just make sure everything you install is on the approved mods list.

6. Join the Community

The best way to learn the ropes, find allies, and stay informed about server news is to join our Discord server. The community is active, and you can ask questions, look for group members, or just chat about Minecraft.

Troubleshooting Connection Issues

If you are having trouble connecting, here are the most common fixes:

  • Wrong version: Make sure you are on Minecraft 1.21.11 for Java Edition. Using a different version will result in a connection error.
  • Incorrect server address: Double-check that you typed refinedvanilla.net exactly, with no extra spaces or characters.
  • Firewall or network issues: Some school or corporate networks block Minecraft server connections. Try connecting from a different network or using a VPN.
  • Bedrock port: Make sure you are using port 19132 for Bedrock Edition.
  • Still stuck? Head to our Discord and ask for help in the support channel.
