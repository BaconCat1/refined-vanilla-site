The Most Vanilla Anarchy Server

RefinedVanilla is a vanilla anarchy survival Minecraft server running version 1.21.11 with full Java + Bedrock cross-play support. We use a custom server software based on Fabric and designed to be as vanilla as possible, with multithreading to eliminate lag — making us the most vanilla server out there.

Unlike typical anarchy servers, RefinedVanilla enforces strict no hacks, no cheats rules. Hacked clients and unfair modifications are banned. We allow TNT, rail, and carpet duping, but all other exploits are off the table. The result is a pure anarchy survival Minecraft experience where skill and vanilla mechanics decide the outcome.

Ready to play? Join RefinedVanilla today, or connect with the community on Discord.
